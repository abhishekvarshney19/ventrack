# ventrack
Vendor Performance Analytics Dashboard built using Python, SQL, and Power BI. Includes automated ETL pipeline, large-scale data ingestion (12M+ rows), vendor sales summary modeling, KPI calculations, and an interactive dashboard for vendor, brand, and profit insights.
